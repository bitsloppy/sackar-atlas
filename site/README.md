# Site

Astro static site — the public-facing queer-heritage website.

**Licence:** MIT — see [../LICENSE-code](../LICENSE-code)

---

## Stack

- **Framework:** [Astro](https://astro.build) (static site generator, markdown-native)
- **Map:** Leaflet + OpenStreetMap (no API key, no vendor lock-in)
- **Search:** Pagefind (static, built at deploy time)
- **Hosting:** Cloudflare Pages (free tier, auto-deploys on push)
- **Database:** SQLite generated at build time from `data/`
